
# **KMP 算法与 NextVal 数组动态可视化工具**

这是一个基于原生 HTML5/JavaScript 和 Tailwind CSS 开发的交互式学习工具。它旨在通过动态动画直观地展示 KMP 字符串匹配算法的核心逻辑，特别是 **Next 数组的推导**、**NextVal 数组的优化**以及**最终的匹配过程**。

## **🌟 核心特性**

* **分步演示**：提供前进/后退控制，精确观察算法指针的每一次移动。  
* **三种模式**：  
  * **Next 数组生成**：展示如何通过“相等最长前后缀”构建基础跳转表。  
  * **NextVal 优化**：对比普通 Next 数组，展示如何通过“字符继承”减少重复失配。  
  * **KMP 匹配演示**：在主串中模拟匹配过程，直观感受“大跨度跳转”带来的效率提升。  
* **实时反馈**：每一步都有对应的中文逻辑解释，辅助理解算法细节。  
* **响应式设计**：完美适配移动端和桌面端。

## **核心算法原理简述**

### **1\. Next 数组**

Next 数组（前缀函数）记录了模式串中每一个子串的“相等且最长的真前缀和真后缀”的长度。它决定了在不匹配时，模式串指针 j 该跳到哪里。

### **2\. NextVal 优化**

这是对 KMP 的进一步改良。如果 p\[i\] \== p\[next\[i\]\]，说明跳转后的位置字符与当前失配字符相同，必然会再次导致失配。nextval 会让指针直接跳向更上层的不同字符，从而消除多余的比较步骤。
